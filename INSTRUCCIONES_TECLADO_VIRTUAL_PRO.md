# Teclado Virtual JEA PRO - Actualización

## Archivos incluidos
- `index.html`
- `manifest.webmanifest`
- `sw.js`

## Cómo subirlo a GitHub Pages
1. Entra al repositorio `Teclado_Virtual`.
2. Reemplaza `index.html`.
3. Reemplaza `manifest.webmanifest`.
4. Agrega `sw.js` en la raíz del repositorio.
5. Conserva los iconos actuales:
   - `android-chrome-192x192.png`
   - `android-chrome-512x512.png`
   - `apple-touch-icon.png`
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `maskable-512x512.png`
6. Haz commit en `main`.
7. Abre GitHub Pages y recarga con Ctrl + F5.
8. En celular, elimina el acceso anterior y vuelve a agregar la app a pantalla de inicio si todavía ves el icono viejo.

## Mejoras principales
- Teclado configurable de 25, 37, 49, 61 y 88 teclas.
- Notas Do-Re-Mi o C-D-E.
- Transposición.
- Sonidos tipo piano, órgano, synth, bajo y campana.
- Acordes y escalas con resaltado visual.
- Metrónomo con tap tempo y subdivisiones.
- Canciones de práctica.
- Grabadora interna de notas.
- Quiz de notas, oído y acordes.
- Soporte básico para Web MIDI cuando el navegador lo permite.
- PWA offline con Service Worker separado.
